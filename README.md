# 🎌 Catálogo Dinâmico de Animes - ADS

Este projeto foi desenvolvido como atividade prática para a disciplina do **Professor Heber Castro**, focando na aplicação de conceitos de **Memória e Lógica** em JavaScript.

## 🚀 Tecnologias Utilizadas
* **HTML5**: Estruturação semântica da página[cite: 3, 5].
* **CSS3**: Estilização moderna com efeitos de Glassmorphism e Blur[cite: 2, 5].
* **JavaScript (ES6+)**: Manipulação de DOM, lógica de programação e gerenciamento de estado[cite: 1, 5].

## 🧠 Conceitos Aplicados
O projeto demonstra o uso técnico de:
* **Memória**: Armazenamento de dados em um Array de Objetos contendo 15 animes.
* **Lógica do Motor**:
    * **Portas Lógicas (&&)**: Destaque visual (Classe `.premium`) para animes com nota > 9 e mais de 100 episódios.
    * **Operador Ternário**: Utilizado para gerenciar a interface de acordo com a idade do usuário.
* **Dinâmica de Aula (Censura)**: Implementação de filtro `blur` via CSS e bloqueio de informações para usuários abaixo da faixa etária permitida[cite: 2, 3].

## 📂 Estrutura do Projeto
* `index.html`: Estrutura principal[cite: 3].
* `style.css`: Estilização e efeitos visuais[cite: 2].
* `script.js`: Banco de dados (15 itens) e lógica funcional[cite: 1].
* `imagens/`: Pasta contendo os arquivos de mídia locais.

## 🛠️ Como executar
1. Clone o repositório.
2. Abra o arquivo `index.html` em seu navegador.
3. Insira sua idade no prompt inicial para navegar no catálogo.
